## Installation

There is one recommended way to install `horstoeko/mimedb` via [Composer](https://getcomposer.org/):

* adding the dependency to your ``composer.json`` file:

```js
  "require": {
      ..
      "horstoeko/mimedb":"^1",
      ..
  },
```
